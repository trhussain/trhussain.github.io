Objective

The objective of this testing plan is to thoroughly test the PIG robot’s capability to navigate through pipeline tunnels, detect pipe cracks, and provide real-time feedback to a GUI for operator monitoring and control. The testing will ensure the robot’s reliability, durability, and accuracy in pipeline inspection scenarios.

Movement and Navigation Testing

Straight-Line Navigation

Objective: Ensure the robot can move in a straight line in a XXXXXX-meter pipeline section.

Success Criteria: The robot moves in a straight path without veering.

Turning and Bend Navigation

Objective: Verify that the robot can navigate 45°, 90°, and 180° bends.

Success Criteria: The robot smoothly navigates each turn without stopping for to long.

Incline and Decline Movement

Objective: Test the robot’s ability to navigate a 15° incline and decline within the pipeline.

Success Criteria: The robot maintains stability while ascending and descending.

Crack Detection Testing

Ultrasonic Sensor Calibration

Objective: Calibrate sensors to detect cracks within ±30 mm accuracy.

Success Criteria: Sensors accurately sense fissures.

Real-Time Data Feedback to GUI

Objective: Ensure crack detection data is accurately transmitted to the GUI in real-time.

Success Criteria: Video feed appears on the GUI with less than XXXX-second delay.

GUI Testing

GUI Display of Real-Time Data

Objective: Ensure all sensor data and video feed are displayed on the GUI.

Success Criteria: GUI displays data accurately with less than 1-second lag.
